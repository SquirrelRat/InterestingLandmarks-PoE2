# Interesting Landmarks for PoE2

A plugin that highlights important objects like strongboxes, league mechanics, quests, and essences directly on the map.

---

## Features

* **Comprehensive Landmark Display:** Shows chests, valuable strongboxes, essences, breaches, rituals, shrines, area transitions, and more.

* **Marker Clustering:** Automatically groups nearby icons into a single label with a counter to keep your map clean.

* **Distance-Based Fading:** Fades out common chest labels and their backgrounds as you move away from them.
